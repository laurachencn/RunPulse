Audio Cues

Customizable spoken alerts during runs — pace, HR, distance milestones, intervals. Configurable triggers and voice messages.

B

Run Goals

Set targets before or during a run — distance, time, pace, calories. Real-time progress feedback and completion celebrations.

C

Trends & Analytics

Weekly/monthly charts, pace trends, HR zones, personal records. iOS dashboard with visual graphs and insights.

D

Auto-Pause

Automatically pause when stopped (traffic lights, water breaks). Resume when movement detected. Smart detection with configurable sensitivity.
